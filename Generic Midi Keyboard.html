GENERIC MIDI KEYBOARD

This script is a simple Midi input script to be used with any midi keyboard without additional controls or functions.

Just add it manually to the controller list and set up the input/output of your Midi interface or your USB Midi Keyboard in order to use it to play instrument devices in Bitwig Studio.
